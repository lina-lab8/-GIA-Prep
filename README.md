# -GIA-Prep

[Русский](#русский) / [English](#english)

## Русский

Веб-приложение для подготовки к ГИА по юриспруденции. 
Всё нужное для самоподготовки в одном месте: материал, тесты и личный прогресс

### Возможности

- **Личный кабинет** — авторизация, у каждого свои отметки о пройденных тестах и материале
- **Вопросы и решения** — ответы на вопросы ГИА и готовые решения задач с пояснениями
- **Видео** — ссылки на лекции, разборы и краткие справки (в основном с YouTube), открываются прямо на сайте
- **Тесты** — проверка знаний по вопросам ГИА
- **Карточки** — для запоминания и быстрого повторения
- **Заметки** — личные пометки по материалу

### Стек

- **Frontend:** HTML, CSS, JavaScript
- **Backend / БД / авторизация:** [Supabase](https://supabase.com)
- **Хостинг:** Netlify / Vercel

### Статус

в разработке


## English

A web app for preparing for the law state exam (ГИА)
Everything you need for self-study in one place: materials, tests and personal progress.

### Features

- **Personal dashboard** — authentication; each user tracks their own completed tests and materials
- **Q&A and solutions** — answers to exam questions and worked-out problem solutions with explanations
- **Videos** — links to lectures, walkthroughs and quick references (mostly from YouTube), embedded in the app
- **Tests** — self-checks based on exam questions
- **Flashcards** — for memorization and quick review
- **Notes** — personal notes on the material

### Tech stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend / DB / auth:** [Supabase](https://supabase.com)
- **Hosting:** Netlify / Vercel

### Status

in progress
